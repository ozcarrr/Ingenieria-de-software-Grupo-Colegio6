# Código Fuente — Kairos

Este directorio contiene el código fuente completo del proyecto Kairos, depurado y sin dependencias pesadas.

## Estructura

```
codigo_fuente/
├── backend/    # API REST — ASP.NET Core 8 (Clean Architecture)
└── frontend/   # Aplicación Flutter (web + mobile)
```

El código fuente completo se encuentra en las carpetas `backend/` y `frontend/` de este directorio.

Para instrucciones de ejecución, consultar el [README principal](../README.md).
